# Ray Tracer

Building a fully functional ray tracer from scratch based on the "Ray Tracing" series by Peter Shirley.

### Completed Features

- Spheres and Light Rays
- Antialiasing
- Lambertian, Metal, and Dielectric Materials
- Positionable Camera
- Defocus Blur
- Motion Blur
- Bounding Volume Hierarchy (BVH)
